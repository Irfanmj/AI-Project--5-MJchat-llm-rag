# MJ Chat — AI RAG Assistant

An AI chatbot built with FastAPI + LLM + Retrieval-Augmented Generation + Session Memory.

## Features
- Conversational memory
- Context-aware responses
- FastAPI backend
- Custom frontend UI
- Session tracking
- RAG architecture

## Tech Stack
- Python
- FastAPI
- Transformers
- Vanilla JS
- HTML/CSS

## Run Locally

```bash
uvicorn backend.app:app --reload
```

Open:
http://127.0.0.1:8000
